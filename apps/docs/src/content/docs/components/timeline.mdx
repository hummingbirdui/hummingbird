## Overview

The Timeline component is used to display a sequence of events in chronological order. It supports vertical and horizontal layouts, color states, and custom indicators.

---

## Anatomy

A timeline consists of:

- `timeline` → root container
- `timeline-item` → single event
- `timeline-dot` → indicator node
- optional start / end content

## Variants

### Centered
Centers content around the timeline axis.

### Opposite
Places content on alternating sides.

### Flush
Removes spacing between items for compact layouts.

---

## Best Practices

- Keep timeline items short and readable
- Use icons sparingly for key milestones only
- Prefer vertical layout for dense information
- Use horizontal layout for step flows or onboarding
