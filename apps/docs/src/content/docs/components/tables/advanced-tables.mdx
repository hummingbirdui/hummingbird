<HbAlert type="info">The Advanced Tables component uses the headless (unstyled) <a href="https://tabulator.info/" target="_blank">Tabulator</a> plugin for its core functionality, while Hummingbird handles the visual styling. Please refer to the official <a href="https://tabulator.info/docs/6.3/quickstart" target="_blank">Tabulator documentation</a> for initialization instructions.</HbAlert>

## How to use

#### 1. Install tabulator

<Example lang="bash" showPreview={false} code={`npm install tabulator-tables --save`} />

#### 2. Import CSS

Import styles after importing Tailwind and Hummingbird CSS:

<Example showPreview={false} lang='css'
  code={`@import '@hummingbirdui/hummingbird/src/plugins/tabulator.css';`} />

For more options and configuration guidance, refer to the <a href="https://tabulator.info/examples/6.3" target="_blank">Tabulator examples</a>.
