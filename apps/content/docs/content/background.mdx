## Background image

Use `bg-*` utilities to apply a background image to an element.

<Example
  code={`<div class="h-48 w-full bg-[url('/images/mountain.avif')] bg-cover bg-bottom rounded-lg"></div>`}
/>

## Linear gradient

Use `bg-gradient-*` or `bg-linear-*` utilities to create linear gradients.

<Example class="space-y-4" code={`<div class="h-14 bg-gradient-to-r from-cyan-500 to-blue-500 rounded-lg"></div>
  <div class="h-14 bg-linear-120 from-emerald-400 to-green-600 rounded-lg"></div>
  <div class="h-14 bg-linear-to-r from-rose-500 to-yellow-500 rounded-lg"></div>
  <div class="h-14 bg-linear-to-t from-purple-500 to-pink-500 rounded-lg"></div>`}
/>

## Radial gradient

Use `bg-radial` utilities to create circular or elliptical gradients.

<Example class="flex items-center justify-center gap-10" code={`<div class="size-24 rounded-full bg-radial from-pink-400 from-40% to-fuchsia-700"></div>
  <div class="size-24 rounded-full bg-radial-[at_50%_75%] from-sky-200 via-blue-400 to-indigo-900 to-90%"></div>
  <div class="size-24 rounded-full bg-radial-[at_25%_25%] from-white to-zinc-900 to-75%"></div>`}
/>

## Conic gradient

Apply `bg-conic` utilities to generate conic (angular) gradients.

<Example class="flex items-center justify-center gap-10" code={`<div class="size-24 rounded-full bg-conic from-blue-600 to-sky-400 to-50%"></div>
  <div class="size-24 rounded-full bg-conic-180 from-indigo-600 via-indigo-50 to-indigo-600"></div>
  <div class="size-24 rounded-full bg-conic/decreasing from-violet-700 via-lime-300 to-violet-700"></div>`} 
/>

## Background image with overlay

Combine a background image with an overlay by stacking elements and using opacity utilities.

<Example code={`<div class="relative h-48 w-full bg-[url('/images/mountain.avif')] bg-cover bg-bottom rounded-lg">
    <div class="absolute inset-0 bg-black/40 rounded-lg"></div>
    <div class="relative z-10 flex h-full items-center justify-center text-white text-2xl font-semibold">
      Background Image
    </div>
  </div>`}
/>


## Background size

Use `bg-contain` to fit the entire image inside the container, or `bg-cover` to fill the container while preserving aspect ratio.

<Example showMarkup={false} class="space-y-6" code={`<div class="max-w-92 mx-auto">
    <p class="mb-1"><code>.bg-contain</code></p>
    <div class="h-40 w-full bg-[url('/images/mountain.avif')] bg-bottom bg-contain bg-no-repeat border border-base"></div>
  </div>
  <div class="max-w-92 mx-auto">
    <p class="mb-1"><code>.bg-cover</code></p>
    <div class="h-40 w-full bg-[url('/images/mountain.avif')] mx-auto bg-bottom bg-cover bg-no-repeat border border-base"></div>
  </div>`}
/>

<Example showPreview={false} code={`<!-- bg contain -->
  <div class="bg-[url('/images/mountain.avif')] bg-contain ..."></div>

  <!-- bg cover -->
  <div class="bg-[url('/images/mountain.avif')] bg-cover ..."></div>`} 
/>
