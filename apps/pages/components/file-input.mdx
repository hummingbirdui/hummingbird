### Sizing

Set heights using classes like <code>.form-control-sm</code> and <code>.form-control-lg</code> for small and large File Input. Default is medium.

<Example>
  <div slot="preview" class="not-prose grid gap-3">
    <input type="file" class="form-control-fill form-control-sm" />
    <input type="file" class="form-control-fill" />
    <input type="file" class="form-control-fill form-control-lg" />
    <input type="file" class="form-control form-control-sm" />
    <input type="file" class="form-control" />
    <input type="file" class="form-control form-control-lg" />
  </div>
  <div slot="code">
    ```html
    <input type="file" class="form-control-fill form-control-sm" />
    <input type="file" class="form-control-fill" />
    <input type="file" class="form-control-fill form-control-lg" />
    <input type="file" class="form-control form-control-sm" />
    <input type="file" class="form-control" />
    <input type="file" class="form-control form-control-lg" />
    ```
  </div>
</Example>

### Color Variants

Different colors become visible while the input is focused. By default, the file input uses the **primary** color. Use additional utility classes to modify the focus color and the file input button color. Like:

- **`.form-control-secondary`**
- **`.form-control-success`**

<Example>
  <div slot="preview" class="not-prose grid items-center gap-3 sm:grid-cols-2">
    <input type="file" class="form-control-fill form-control-secondary" />
    <input type="file" class="form-control-fill form-control-success" />
    <input type="file" class="form-control form-control-secondary" />
    <input type="file" class="form-control form-control-success" />
  </div>
  <div slot="code">
    ```html
    <input type="file" class="form-control-fill form-control-secondary" />
    <input type="file" class="form-control-fill form-control-success" />
    <input type="file" class="form-control form-control-secondary" />
    <input type="file" class="form-control form-control-success" />
    ```
  </div>
</Example>

### Validation

Indicate validation states by adding the `.is-invalid` class to file input. This highlights the input field when invalid or requiring user attention.

<Example>
  <div slot="preview" class="not-prose grid items-center gap-3 sm:grid-cols-2">
    <div class="form-field">
      <label class="form-label">Avatar</label>
      <input type="file" class="form-control-fill is-invalid" />
    </div>
    <div class="form-field">
      <label class="form-label">Avatar</label>
      <input type="file" class="form-control is-invalid" />
    </div>
  </div>
