## Installation

#### 1. Install Tailwind CSS
Follow the <a href="https://tailwindcss.com/docs/installation/framework-guides/nuxt" target="_blank" rel="noopener noreferrer">official installation guide</a> to install Tailwind CSS in your Nuxt project.

#### 2. Install Hummingbird

Install Hummingbird via your preferred package manager:

<Example lang='bash' showPreview={false} code={`npm install @hummingbirdui/hummingbird`} />

#### 3. Import CSS

Import Hummingbird styles in your main CSS file (e.g., `main.css`).

<Example lang='css' showPreview={false} 
  code={`@import "tailwindcss";
  @import "@hummingbirdui/hummingbird";`} />

#### 4. Initialize JS plugins

#### Composable

Create a composable `useHummingbird.ts` in your `composables` directory, this composable ensures Hummingbird scripts run only after the Nuxt app has mounted and hydration is complete.

<Example lang='js' showPreview={false}
  code={`import { onMounted } from "vue";

  export function useHummingbird(callback: () => void) {
    const nuxtApp = useNuxtApp();

    onMounted(() => {
      if (import.meta.client && !nuxtApp.isHydrating) {
        callback();
      }
    });
  }`} />

Import and call `useHummingbird` inside a Nuxt page (e.g., `app.vue` file) to load and initialize the Hummingbird JavaScript bundle.

<Example lang='vue' showPreview={false}
  code={`<script setup>
  import { useHummingbird } from './composable/useHummingbird';

  useHummingbird(() => {
    import('@hummingbirdui/hummingbird');
  });
  </script>

  <template>
  ...
  </template>`} />

## JavaScript API

Hummingbird also provides a JavaScript API for programmatic control of components. For example, you can toggle a modal manually using the API.

Hereâ€™s how to initialize and use the [Modal](/docs/components/modal) component:

<Example lang='vue' showPreview={false}
  code={`<script setup>
  import { useHummingbird } from './composable/useHummingbird';

  useHummingbird(async () => {
    const { Modal } = await import('@hummingbirdui/hummingbird');

    const myModal = new Modal('.modal')
    const openBtn = document.querySelector('[data-open-demo-modal]');

    openBtn.addEventListener('click', () => {
      myModal.toggle()
    })
  });
  </script>

  <template>
    <button class='btn btn-primary' data-open-demo-modal>Open</button>
    <div class="modal">...</div>
  </template>`} />
