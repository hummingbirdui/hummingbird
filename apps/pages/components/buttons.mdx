### Basic Buttons

The Button comes with different variants

### Contained button

Contained buttons are high-emphasis, distinguished by their use of elevation and fill. They contain actions that are primary to your app.

<Example>
  <div slot="preview" class='not-prose space-x-2'>
    <button class='btn-primary'>Primary</button>
    <button class='btn-secondary'>Secondary</button>
    <button class='btn-info'>Info</button>
    <button class='btn-success'>Success</button>
    <button class='btn-warning'>Warning</button>
    <button class='btn-error'>Error</button>
    <button class='btn-neutral'>Neutral</button>
  </div>

  <div slot="code">
  ```html
  <button class='btn-primary'>Primary</button>
  <button class='btn-secondary'>Secondary</button>
  <button class='btn-info'>Info</button>
  <button class='btn-success'>Success</button>
  <button class='btn-warning'>Warning</button>
  <button class='btn-error'>Error</button>
  <button class='btn-neutral'>Neutral</button>
  ```
  </div>
</Example>

### Subtle button

Subtle buttons provide a gentle emphasis with a subtle background and solid text color, ideal for secondary actions that need visual distinction without overpowering the primary content.

<Example>
  <div slot="preview" class='not-prose space-x-2'>
    <button class='btn-primary-subtle'>Primary</button>
    <button class='btn-secondary-subtle'>Secondary</button>
    <button class='btn-info-subtle'>Info</button>
    <button class='btn-success-subtle'>Success</button>
    <button class='btn-warning-subtle'>Warning</button>
    <button class='btn-error-subtle'>Error</button>
    <button class='btn-neutral-subtle'>Neutral</button>
  </div>

  <div slot="code">
  ```html
  <button class='btn-primary-subtle'>Primary</button>

  <button class='btn-warning-subtle'>Warning</button>
  <button class='btn-error-subtle'>Error</button>
  <button class='btn-neutral-subtle'>Neutral</button>
  ```
  </div>
</Example>

Outlined buttons are medium-emphasis buttons. They contain actions that are important but aren't the primary action in an app.

    <button class='btn-secondary-outline'>Secondary</button>

### Text button

Text buttons are typically used for less-pronounced actions, including those located: in dialogs, in cards. In cards, text buttons help maintain an emphasis on card content.

<Example>

    <button class='btn-info-text'>Info</button>

    <button class='btn-warning-text'>Warning</button>

### Button Sizes

For larger or smaller buttons, use the <code>.btn-lg</code> and <code>.btn-sm</code> class. Default is medium.

<Example>
  <div slot="preview" class='not-prose space-y-3'>
    <div class='space-x-2'>
      <button class='btn-primary btn-sm'>Small</button>
      <button class='btn-primary'>Medium</button>
      <button class='btn-primary btn-lg'>Large</button>
    </div>
    <div class='space-x-2'>
      <button class='btn-primary-subtle btn-sm'>Small</button>
      <button class='btn-primary-subtle'>Medium</button>
      <button class='btn-primary-subtle btn-lg'>Large</button>
    </div>
    <div class='space-x-2'>
      <button class='btn-primary-outline btn-sm'>Small</button>
      <button class='btn-primary-outline'>Medium</button>
      <button class='btn-primary-outline btn-lg'>Large</button>
    </div>
    <div class='space-x-2'>
      <button class='btn-primary-text btn-sm'>Small</button>
      <button class='btn-primary-text'>Medium</button>
      <button class='btn-primary-text btn-lg'>Large</button>
    </div>
  </div>

  <div slot="code">
  ```html
  <button class='btn-primary btn-sm'>Small</button>
  <button class='btn-primary'>Medium</button>
  <button class='btn-primary btn-lg'>Large</button>
  <button class="btn-primary-subtle btn-sm">Small</button>
  <button class="btn-primary-subtle">Medium</button>
  <button class="btn-primary-subtle btn-lg">Large</button>
  <button class='btn-primary-outline btn-sm'>Small</button>
  <button class='btn-primary-outline'>Medium</button>
  <button class='btn-primary-outline btn-lg'>Large</button>
  <button class='btn-primary-text btn-sm'>Small</button>

  ```
  </div>
</Example>
