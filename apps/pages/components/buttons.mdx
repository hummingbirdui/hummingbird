### Basic Buttons

The Button comes with different variants

### Contained button

Contained buttons are high-emphasis, distinguished by their use of elevation and fill. They contain actions that are primary to your app.

<Example>
  <div slot="preview" class='not-prose space-x-2 space-y-2'>
    <button class='btn btn-primary'>Primary</button>
    <button class='btn btn-secondary'>Secondary</button>
    <button class='btn btn-info'>Info</button>
    <button class='btn btn-success'>Success</button>
    <button class='btn btn-warning'>Warning</button>
    <button class='btn btn-danger'>Danger</button>
    <button class='btn btn-neutral'>Neutral</button>
  </div>

  <div slot="code">
  ```html
  <button class='btn btn-primary'>Primary</button>
  <button class='btn btn-secondary'>Secondary</button>
  <button class='btn btn-info'>Info</button>
  <button class='btn btn-success'>Success</button>
  <button class='btn btn-warning'>Warning</button>
  <button class='btn btn-danger'>Danger</button>
  <button class='btn btn-neutral'>Neutral</button>
  ```
  </div>
</Example>

### Subtle button

Subtle buttons provide a gentle emphasis with a subtle background and solid text color, ideal for secondary actions that need visual distinction without overpowering the primary content.

<Example>
  <div slot="preview" class='not-prose space-x-2 space-y-2'>
    <button class='btn btn-subtle-primary'>Primary</button>
    <button class='btn btn-subtle-secondary'>Secondary</button>
    <button class='btn btn-subtle-info'>Info</button>
    <button class='btn btn-subtle-success'>Success</button>
    <button class='btn btn-subtle-warning'>Warning</button>
    <button class='btn btn-subtle-danger'>Danger</button>
    <button class='btn btn-subtle-neutral'>Neutral</button>
  </div>

  <div slot="code">
  ```html
  <button class='btn btn-subtle-primary'>Primary</button>
  <button class='btn btn-subtle-secondary'>Secondary</button>
  <button class='btn btn-subtle-info'>Info</button>

  <button class='btn btn-subtle-warning'>Warning</button>
  <button class='btn btn-subtle-danger'>Danger</button>
  <button class='btn btn-subtle-neutral'>Neutral</button>
  ```
  </div>
</Example>

Outlined buttons are medium-emphasis buttons. They contain actions that are important but aren't the primary action in an app.

### Text button

Text buttons are typically used for less-pronounced actions, including those located: in dialogs, in cards. In cards, text buttons help maintain an emphasis on card content.

<Example>

    <button class='btn btn-text-secondary'>Secondary</button>

    <button class='btn btn-text-warning'>Warning</button>
    <button class='btn btn-text-danger'>Danger</button>
    <button class='btn btn-text-neutral'>Neutral</button>

### Button Sizes

For larger or smaller buttons, use the <code>.btn-lg</code> and <code>.btn-sm</code> class. Default is medium.

<Example>
  <div slot="preview" class='not-prose space-y-3'>
    <div class='space-x-2'>
      <button class='btn btn-primary btn-sm'>Small</button>
      <button class='btn btn-primary'>Medium</button>
      <button class='btn btn-primary btn-lg'>Large</button>
    </div>
    <div class='space-x-2'>
      <button class='btn btn-subtle-primary btn-sm'>Small</button>
      <button class='btn btn-subtle-primary'>Medium</button>
      <button class='btn btn-subtle-primary btn-lg'>Large</button>
    </div>
    <div class='space-x-2'>
      <button class='btn btn-outline-primary btn-sm'>Small</button>
      <button class='btn btn-outline-primary'>Medium</button>
      <button class='btn btn-outline-primary btn-lg'>Large</button>
    </div>
    <div class='space-x-2'>
      <button class='btn btn-text-primary btn-sm'>Small</button>
      <button class='btn btn-text-primary'>Medium</button>
      <button class='btn btn-text-primary btn-lg'>Large</button>
    </div>
  </div>

  <div slot="code">

</Example>
