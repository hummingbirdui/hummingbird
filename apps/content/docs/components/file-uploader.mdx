<HbAlert type="info">The File Uploader component uses the  <a href="https://www.dropzone.dev/" target="_blank" class="text-default no-underline">**Dropzone.js**</a> plugin for its core functionality, while Hummingbird handles the visual styling. Please refer to the official <a href="https://docs.dropzone.dev/" target="_blank">Dropzone.js documentation</a> for initialization instructions.</HbAlert>

## How to use

#### 1. Install Dropzone.js

<Example lang="bash" showPreview={false} code={`$ npm install --save dropzone`} />

#### 2. Import CSS

Import styles after importing Tailwind and Hummingbird CSS:

<Example showPreview={false} lang='css'
  code={`@import '@hummingbirdui/hummingbird/src/plugins/dropzone.css';`} />

#### 3. Initialization

<Example showPreview={false} lang='js'
  code={`let myDropzone = new Dropzone("div#myId", { url: "/file/post"});`} />
