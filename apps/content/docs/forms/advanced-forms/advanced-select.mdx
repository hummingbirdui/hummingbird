<HbAlert type="info">The Advance Select component uses the headless (unstyled) <a href="https://choices-js.github.io/Choices/" target="_blank">Choices.js</a> plugin for its core functionality, while Hummingbird handles the visual styling. Please refer to the official <a href="https://github.com/Choices-js/Choices" target="_blank">Choices.js documentation</a> for initialization instructions.</HbAlert>

## How to use

#### 1. Install choices.js

<Example lang="bash" showPreview={false} code={`npm install choices.js`} />

#### 2. Import CSS

Import styles after importing Tailwind and Hummingbird CSS:

<Example showPreview={false} lang='css'
  code={`@import '@hummingbirdui/hummingbird/src/plugins/choices.css';`} />

#### 3. Initialization

Basic initialization example with remove item button enabled:

<Example showPreview={false} lang='js'
  code={`const element = document.querySelector('[data-choices="data-choices"]');
  const choices = new Choices(element, { removeItemButton: true });`} />

For more options and configuration guidance, refer to the <a href="https://github.com/Choices-js/Choices" target="_blank">Choices.js documentation</a>.

## Text Input

Text input with Choices.js. Functionality can be extended using Choices.js options.

<Example code={`<input type="text" class='form-control' id="name" name="name" data-choices="data-choices" />`} />

## Multiple select input

Select input with multiple selection enabled by adding `multiple` attribute.

<Example code={`<select class="form-select" aria-label="Outlined example with label" data-choices="data-choices" multiple>
    <option value="">Select organizer...</option>
    <option>Massachusetts Institute of Technology</option>
    <option>University of Chicago</option>
    <option>GSAS Open Labs At Harvard</option>
    <option>California Institute of Technology</option>
  </select>`} />

## Single select input

Select single value by adding `select-one` attribute.

<Example code={`<select class="form-select" aria-label="Outlined example with label" data-choices="data-choices" select-one>
    <option value="">Select organizer...</option>
    <option>Massachusetts Institute of Technology</option>
    <option>University of Chicago</option>
    <option>GSAS Open Labs At Harvard</option>
    <option>California Institute of Technology</option>
  </select>`} />

## Customization

### CSS Variables

Advanced select component inherits styles from `form-control` and `form-select` classes. Any CSS variables defined for these classes can be overridden to customize the appearance of the advance select component. Follow the [form control](/docs/components/form-control#css-variables) and [select](/docs/components/select#css-variables) documentation for available CSS variables.

### Choices.js Classes

Customization also can be done via overriding Choices.js default classes.
