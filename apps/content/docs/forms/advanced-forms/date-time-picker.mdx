<HbAlert type="info">The Date & Time picker component uses the  <a href="https://flatpickr.js.org/" target="_blank">flatpickr.js</a> plugin for its core functionality, while Hummingbird handles the visual styling. Please refer to the official <a href="https://flatpickr.js.org/getting-started/" target="_blank">flatpickr.js documentation</a> for initialization instructions.</HbAlert>

## How to use

#### 1. Install flatpickr.js

<Example lang="bash" showPreview={false} code={`$ npm i flatpickr --save`} />

#### 2. Import CSS

Import styles after importing Tailwind and Hummingbird CSS:

<Example showPreview={false} lang='css'
  code={`@import '@hummingbirdui/hummingbird/src/plugins/flatpickr.css';`} />

#### 3. Initialization

<Example showPreview={false} lang='js'
  code={`import flatpickr from "flatpickr";

  // If using flatpickr in a framework, its recommended to pass the element directly
  flatpickr(element, {});

  // Otherwise, selectors are also supported
  flatpickr("#myID", {});

  // creates multiple instances
  flatpickr(".anotherSelector");`}
/>

## Disabled dates

<Example class="mx-auto max-w-98" code={`<input id="disabled-dates" type="text" class="form-control" placeholder="dd-mm-yyyy" />`} />
<Example lang='js' showPreview={false}
  code={`flatpickr('#disabled-dates', {
    dateFormat: 'd-m-Y',
    disable: [
      function (date) {
        return date.getDay() === 5 || date.getDay() === 6;
      },
    ],
  });`}
/>

## Highlight weekends

<Example class="mx-auto max-w-98" code={`<input id="highlight-weekend" type="text" class="form-control" placeholder="dd-mm-yyyy" />`} />
<Example lang='js' showPreview={false}
  code={`flatpickr('#highlight-weekend', {
    dateFormat: 'd-m-Y',
    onDayCreate: function (dObj, dStr, fp, dayElem) {
      const day = dayElem.dateObj.getDay();

      if (day === 5 || day === 6) {
        dayElem.classList.add('weekend-day');
      }
    },
  });`}
/>
