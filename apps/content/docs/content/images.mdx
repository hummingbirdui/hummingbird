## Basic example

Embed images using the standard HTML `<img>` tag and apply Tailwind CSS utility classes for styling.

<Example code={`<img class="w-full rounded-lg" src="${getVersionedPath('/images/2.avif')}" alt="Sample image" />`}/>

## Sizes

Control image size using Tailwind CSS width and height utilities.

<Example code={`<img class="w-76 h-48 rounded-md" src="${getVersionedPath('/images/2.avif')}" alt="Medium image" />`}/>

Use the `size-*` utility to apply equal width and height on an image.

<Example code={`<img class="size-48 object-cover rounded-md" src="${getVersionedPath('/images/2.avif')}" alt="Medium image" />`}/>

## Aspect ratio

Use Tailwindâ€™s `aspect-*` utilities to preserve image proportions.

<Example code={`<img class="aspect-3/2 object-cover rounded-lg" src="${getVersionedPath('/images/2.avif')}" alt="3:2 image" />`}/>

### Square aspect ratio

<Example code={`<img class="max-h-64 aspect-square object-cover rounded-lg" src="${getVersionedPath('/images/2.avif')}" alt="Square image"/>`}/>

## Object fit

Control how the image fits within its container using `object-*` utilities.

<Example showMarkup={false} class="flex flex-wrap justify-center gap-6" code={`<figure class="w-72">
    <img
      src="${getVersionedPath('/images/2.avif')}"
      alt="Contain example"
      class="h-40 w-full object-contain border border-base"
    />
    <figcaption class="mt-2 text-sm text-center">Object contain</figcaption>
  </figure>
  
  <figure class="w-72">
    <img
      src="${getVersionedPath('/images/2.avif')}"
      alt="Contain example"
      class="h-40 w-full object-cover border border-base"
    />
    <figcaption class="mt-2 text-sm text-center">Object cover</figcaption>
  </figure>`}
/>

<Example showPreview={false} code={`<!-- object contain -->
  <img class="h-40 w-72 object-contain border border-base" src="${getVersionedPath('/images/2.avif')}" alt="Contain example" />
  
  <!-- object cover -->
  <img class="h-40 w-72 object-cover border border-base" src="${getVersionedPath('/images/2.avif')}" alt="Cover example" />`}
/>

## Custom styles

Apply borders, shadows, and other visual styles using Tailwind CSS.

<Example code={`<img class="p-1 h-64 object-cover bg-default border border-base rounded-md hover:shadow-xl shadow-gray-200 dark:shadow-gray-800 transition duration-300 ease-in-out" src="${getVersionedPath('/images/2.avif')}" alt="Styled image" />`}/>

## Responsive images

Images automatically scale on different screen sizes using responsive utilities.

<Example code={`<img class="w-full sm:w-1/2 lg:w-1/3 rounded-lg" src="${getVersionedPath('/images/2.avif')}" alt="Responsive image" />`}/>

## Accessibility

Always include an `alt` attribute to describe the image content for screen readers and accessibility tools.

<Example code={`<img class="h-64 object-cover rounded-lg" src="${getVersionedPath('/images/2.avif')}" alt="A hummingbird sitting on a flower" />`}/>
