## Image avatar

Image avatars display a user or entity using a photo or graphic.

<Example class='text-center' code={`<div class="avatar">
    <img src="${getVersionedPath('/images/avatar/avatar_1.webp')}" alt="Avatar" class="rounded-full" />
  </div>`}
/>

## Letter avatar

Letter avatars display simple character, to represent a user or entity when an image is not available.

<Example class='flex justify-center gap-2' code={`<div class="avatar">
    <div class="avatar-name rounded-full">
      A
    </div>
  </div>
  <div class="avatar">

    </div>
  </div>
  <div class="avatar">
    <div class="avatar-name bg-warning rounded-full">

  </div>`} />

## Sizes

Avatars support four preset sizes: `.avatar-xs`, `.avatar-sm`, default, and `.avatar-lg`. Custom sizes can also be created by defining height and width.

<Example class='flex items-center justify-center gap-2' code={`<div class="avatar avatar-xs">
    <img src="${getVersionedPath("/images/avatar/avatar_1.webp")}" alt="Avatar" class="rounded-full" />
  </div>
  <div class="avatar avatar-sm">
    <img src="${getVersionedPath("/images/avatar/avatar_2.webp")}" alt="Avatar" class="rounded-full" />
  </div>
  <div class="avatar">
    <img src="${getVersionedPath("/images/avatar/avatar_3.webp")}" alt="Avatar" class="rounded-full" />
  </div>
  <div class="avatar avatar-lg">
    <img src="${getVersionedPath("/images/avatar/avatar_4.webp")}" alt="Avatar" class="rounded-full" />
  </div>`} />

## Avatar text

Avatars can be combined with text to provide additional context, such as user names or roles.

<Example class='flex justify-center' code={`<div class="flex items-center gap-3">
    <div class="avatar avatar-lg">
      <img src="${getVersionedPath("/images/avatar/avatar_3.webp")}" alt="Avatar" class="rounded-full" />
    </div>
    <div>
      <p class="mb-0 font-semibold">Andrew Alfred</p>
      <span class="text-muted">Technical advisor</span>
    </div>
  </div>`}
/>

## Inline avatar

Use small avatars inline with text to highlight users or authors without breaking the content flow.

<Example class='flex justify-center' code={`<p>
    I spent most of the day exploring bottle returns in Michigan. 
    <span class="inline-flex items-baseline gap-1">
      <span class="avatar avatar-xs self-center">
        <img src="${getVersionedPath("/images/avatar/avatar_1.webp")}" alt="Avatar" class="rounded-full" />
      </span>
      <span class="font-medium">Alfred</span>
    </span>
    insists it won’t work, but I still think there’s a way to make it profitable.
  </p>`}
/>

## CSS variables

The avatar component is built using a set of CSS variables. These variables provide flexibility for customizing styles.
See the [Theming](https://hbui.dev/docs/customize/theming/) to customize styles globally and apply overrides as needed.

<Example lang='sass' showPreview={false}
  code={`.avatar {
    --avatar-width: --spacing(10);
    --avatar-height: --spacing(10);
    --avatar-bg: var(--color-dark);
    --avatar-color: var(--color-white);
    --avatar-font-weight: var(--font-weight-medium);
    --avatar-font-size: var(--text-lg);
  }`}
/>
