## Example

Use the `.btn-close` utility to add a dismiss or close action to any component. The default style is minimal and fully customizable with utility classes. Modify the CSS variables to replace the default `background-image`, and include accessible text using `aria-label` for screen readers.

<Example class="flex gap-2"
  code={`<button class="btn-close" aria-label="close"></button>
  <button class="btn-close btn-close-circle" aria-label="close"></button>`}
/>

## Disable state

Disabled close buttons have reduced `opacity` and ignore user interactions, with `pointer-events-none` and `select-none` preventing hover or active states.

<Example 
  code={`<button class="btn-close" aria-label="close" disabled></button>`}
/>

## White variant

Use the `.btn-close-white` class to make the default `.btn-close` white.

<Example class="dark"
  code={`<button class="btn-close btn-close-white" aria-label="close"></button>`}
/>
